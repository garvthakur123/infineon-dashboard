# AI_DISCLOSURE

## AI Tools Used

- Claude Sonnet 4.6

## How AI Was Used

I used Claude Sonnet 4.6 mainly as a coding assistant during the implementation of this take-home challenge.

It was used primarily for:

- frontend UI styling and layout refinement
- design adjustments for the dashboard and detail page
- improving the visual structure of the comment editing flow
- drafting and refining documentation such as the README and supporting files
- occasional error fixing and debugging support during development
- small refactoring suggestions to simplify or clean up parts of the code

## What I Implemented Manually

I handled the following parts myself:

- backend architecture and implementation
- database setup and schema decisions
- MySQL Docker setup
- CSV read flow
- API route structure
- controller and service flow
- comment persistence logic
- merge logic between CSV data and persisted MySQL comments
- frontend data flow
- login flow structure
- frontend state management
- integration between frontend and backend APIs
- overall feature prioritization and scope decisions

## Notes

AI was used as an assistant, not as a replacement for implementation decisions.  
The final structure, architecture, integration flow, and submitted solution were manually reviewed and adjusted by me.
